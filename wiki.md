# Project Summary
DrainCheck NG is a community-driven flood prevention platform aimed at empowering residents of Lagos, Nigeria, to report blocked drains and mitigate flooding risks. Users can quickly submit reports, view community maps of issues, and connect with local cleanup organizations, enhancing civic engagement and public safety.

# Project Module Description
The project consists of four main pages:
1. **Home Page (index.html)**: Introduces the platform, highlights its mission, and provides call-to-action buttons.
2. **Report Issue Page (report.html)**: Users can report blocked drains with a new engaging hero section featuring community reporting themes and a visual guide.
3. **Live Map Page (map.html)**: Displays a community map showing reported issues with enhanced visuals and statistics for better user interaction.
4. **About Page (about.html)**: Shares the story behind the project, its mission, and ways for the community to get involved.

# File Description Inventory
- **about.html**: Contains the project's background, mission, and contact information.
- **index.html**: The landing page with a hero section, statistics, and navigation.
- **map.html**: Interactive map displaying reported drain issues with a legend and recent reports, now includes a new hero section.
- **report.html**: Form for users to submit reports on blocked drains, now enhanced with a new hero section and a visual guide.
- **package.json**: Defines project dependencies and scripts for building and running the project.
- **script.js**: Handles mobile menu interactions, smooth scrolling, and animations.
- **style.css**: Contains styles for layout, colors, typography, and responsive design, including new styles for hero sections.
- **template_config.json**: Configuration file for the project’s template.

# Technology Stack
- **HTML5**: For structuring the web pages.
- **CSS3**: For styling and layout.
- **JavaScript**: For interactivity and dynamic content.
- **Bootstrap 5.3**: For responsive design and UI components.
- **Google Forms**: For data collection on the report page.
- **Google My Maps**: For displaying community reports on the map page.
